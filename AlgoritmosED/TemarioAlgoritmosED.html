Proyecto


Selecciona un bloque de código del lenguaje de alto nivel que quieras. El cuál se traducirá a un lenguaje de bajo nivel usando las etapas de un compilador. Este proyecto se implementará con un máximo de tres Estudiantes.



Forma de llevar el curso


1.- El contenido teórico se impartirá usando esta página, dónde se presenta el temario en el cuál se seleccionará el tema dando click en el enlace correspondiente.


2.- Las prácticas se enlistan en la sección guinda, las que se realizarán durante el curso conforme al avance teórico.


3.- Se presenta una lista de proyectos en la sección guinda. Los que podrán realizar en grupos de tres Estudiantes, como máximo.


4.- Se aplicarán tres examenes, cómo máximo.


Evaluación del curso


1.- Las prácticas y el proyecto sirven para acreditar el curso con 6.


2.- Los examenes servirán para aumentar la calificación final del curso de acuerdo con una tabla que propondré.
