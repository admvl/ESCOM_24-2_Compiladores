REPASO DE DEFINICIONES

Método del algoritmo de subconjuntos

Es un método que tiene como entrada una ER la cuál se transforma en un autómata AFN utilizando la construcción de Thompson este se transforma en un AFD aplicando el algoritmo de subconjuntos el cuál, se minimiza para obtener un autómata AFD mínimo:


ER → AFN → AFD → AFDmínimo


Expresiones regulares


El token se representa por medio de una expresión regular (ER) dónde, se usan las cuatro operaciones básicas:

Token → clase léxica
ER → AFN

Autómatas

Existen dos tipos de autómatas:

Token → clase léxica
ER → AFN → AFD


Autómatas mínimos

Son los que tienen un menor número de estados, para reducir el número de estados se aplica el algoritmo que utiliza el concepto de consistencia:

AFD → AFDmínimo

Método del árbol

Es un método que tiene como entrada una ER y que utiliza un árbol de análisis sintáctico, que tiene como salida un autómata AFD mínimo:


ER → Árbolsintáctico → Funciones → Función siguientepos() → AFDmínimo


Uso de árboles


La ER se posiciona en el árbol de análisis sintáctico:

Símbolos terminales → en las hojas ;
Los operadores → en los nodos padre
