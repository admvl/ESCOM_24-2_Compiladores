CLASES E INSTANCIAS DE CLASE

Objetivo

Se formarán clases a partir de los nombres. Se crearan objetos de cada una de las clases.

Actividades

Las clases permiten que un diseñador de software considere los objetos como diferentes tipos de entidades. Visita objetos de esta manera nos permite utilizar los mecanismos de clasificación para categorizar estos tipos, definir jerarquías y comprometerse con las ideas de especialización y generalización de objetos.


Un objeto es una instancia de clase. Se pueden crear muchas instancias de clase. Crear una instancia de clase se llama instanciación. El término objeto e instancia de clase son intercambiables.


Ejemplo. Compilar y ejecutar el siguiente programa:


       public class Persona {
         String nombre;
         int edad;
         String direccion;
         public void setNombre(String b) {
           nombre = b;
         }
         public void setEdad(int b) {
           edad = b;
         }
         public String getNombre() {
           return nombre;
         }
         public int getEdad() {
           return edad;
         }
         public static void main(String[] args) {
           Persona per = new Persona();
           per.setPersona("Juan ");
           per.setEdad(25);
           System.out.println(per.getNombre() + "tiene " + per.getEdad() + " años");
         }
       }
