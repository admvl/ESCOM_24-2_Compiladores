Sobrecarga

Definición de sobrecarga. En Java es posible definir dos o más métodos dentro de la misma clase que tengan el mismo nombre, pero con sus listas de parámetros distintas. Cuando ocurre esto, se dice que los modos están sobrecargados y a este proceso se le denomina sobrecarga de modo.


Como hace la selección. Cuando se invoca un método sobrecargado, Java utiliza el tipo y/o el número de argumentos como guía para determinar la versión del método sobrecargado que realmente debe llamar.

        class Jarra {
          Jarra (int marcador) {
            System.out.println("Jarra( " + marcador + ") ");
          }
          void f(int marcador) {
            System.out.println("f( " + marcador + ") ");
          }
        }
        public class Jarras {
          Jarra j1;
          Jarra j2;
          {
            j1 = new Jarra(1) ;
            j2 = new Jarra (2) ;
            System.out.println("j1 y j2 inicializadas");
          }
          Jarras() {
            System.out.println ("Jarras() ") ;
          }
          public static void main(String[] args) {
            System.out.println("Dentro de main()");
            Jarras x = new Jarras();
          }
        }

Programa 4.


        public class Tiempo {
          private int horas, minutos, segundos;
          public Tiempo(int h, int m, int s) {
            horas = h;
            minutos = m;
            segundos = s;
          }

          public void imprimirTiempo() {
            System.out.printf("%02d:%02d:%02d\n", horas, minutos, segundos);
          } // fin de imprimirTiempo

          public static void main(String[] args) {
            Tiempo tiempo = new Tiempo(3, 59, 0);
            tiempo.imprimirTiempo();
          } // fin de main
        } // fin de la clase tiempo
