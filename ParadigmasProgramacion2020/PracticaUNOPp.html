Paradigmas de programación
Práctica 1

Temario

FUNCIONES PURAS, DE PRIMER ORDEN Y DE ORDEN SUPERIOR

Objetivo

Se crearan las funciones en diferentes lenguajes de programación.


Actividades


Crear la función pura, de los siguientes problemas:


Problema 1: Presentar un programa que, calcule el volumen de un cilindro: dónde r es su radio del cilindro y L su longitud. Utiliza la siguiente fórmula: V = Pi r2 L.


Problema 2: Presentar un programa que, que calcule la superficie de un cilindro: dónde r es el radio del cilindro y l es la longitud del cilindro. Utiliza la fórmula : 2 Pi r l + 2 Pi r2.


Problema 3: Presentar un programa que, calcule las raices de x del siguiente polinomio; ax2 + bx + c, donde a, b, c son números conocidos.


Cuestionario


Pregunta 1. Sustentar de que tipo son las funciones que se obtienen de los tres problemas anteriores.


Pregunta 2. Adornar las salidas de las tres funciones de los tres problemas anteriores.


Pregunta 3. Escribir en Python las tres funciones de los tres problemas anteriores..


Pregunta 4. Crear una función en Python de orden superior que calcule las tres fórmulas de los tres problemas anteriores.
