CLASE 12: Analizador ascendente LR(0)

¿Que es una analizador ascendente? 

Se tiene la siguiente gramática libre de contexto:






NúmeroProducción



1E → E + T 


2E → T 


3T → x 







Procedimiento de construcción del analizador ascendente LR(0)





Se expande, se aumenta y se divide en capas la gramática


Se obtienen los elementos LR(0)


Se obtiene el grafo de la operación de IR_A y de aplicar la 
continuidad a cada uno de los estados obtenidos


Se obtiene la tabla de acciones y de IR-A, aplicando la tabla 
de primeros y siguientes a los elementos completos


Se analizan sentencias x + x. Se usa una tabla con una pila, 
lectura de símbolos, variable estado y la acción 
que se aplicará
