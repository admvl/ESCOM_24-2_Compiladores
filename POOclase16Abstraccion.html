DEFINICIÓN DE ABSTRACCIÓN



Este principio de POO busca modelar los 
objetos, busca abstraer y simplificar un objeto de la 
vida real a solo un par de atributos. En otras palabras, 
se busca transformar un objeto de la vida real en 
atributos (características) y sus acciones (métodos). 
Consiste en encontrar las partes fundamentales de un 
sistema para describirlas de manera simple y precisa.


Diccionario: 
Real Académia de la lengua


Acción y efecto de abstraer o abstraerse.


DEFINICIÓN DE ABSTRAER


1. tr. Abstraer.- Separar por medio de una operación 
intelectual un rasgo o una cualidad de algo para analizarlos 
aisladamente o considerarlos en su pura esencia o noción. 
U. t. c. intr. Pensar es olvidar diferencias, es 
generalizar, abstraer.

2. intr. Hacer caso omiso de algo, o dejarlo a un lado. 
Centremos la atención en lo esencial abstrayendo DE 
consideraciones marginales. U. t. c. prnl.

3. prnl. Concentrarse en los propios pensamientos 
apartando los sentidos o la mente de la realidad 
inmediata. Aspirando suavemente su cigarro, se 
abstrajo DEL ruido de la disputa.

4. prnl. Retirarse o recogerse, apartándose del 
trato social. Abstraerse DE los negocios mundanos.

DEFINICIÓN DE OBJETO


Un objeto representa una entidad en el mundo real
que se identifica con facilidad. Ejemplos de
 objeto: un estudiante, un escritorio, un círculo,
 un botón, etc.

Un objeto tiene una única identidad, estado, y
 comportamiento.


DEFINICIÓN DE ESTADO DE OBJETO


El estado de un objeto es conocido como su propiedad
o atributo y que es representado por un campo de
datos con su valor actual.

Ejemplo: el objeto circulo, tiene un campo de datos
radio, el cuál es una propiedad que caracteriza
al círculo.

En el caso del rectángulo, el estado del objeto es
el ancho y la altura, lo que caracteriza al
rectángulo.


Comportamiento del
objeto


También llamado su acción y es definido por un
método. Para invocar un método sobre un objeto
se le pregunta al objeto para que desempeñe su
acción.

Ejemplo: para un círculo getArea() se invoca
para regresar el área.


Denotación de objeto


claseNombre objetoRefVar;

claseNombre objetoRefVar = new claseNombre();


Clases


Los objetos del mismo tipo son definidos usando
una clase común. Una clase es una plantilla, plano o 
contrato que define que el campo de datos de un
 objeto y métodos se realicen. Un objeto es una
 instancia de un clase.

Crear una instancia se refiere como
instanciación. El termino objeto e instancia
puede ser intercambiable.


EJEMPLO DE CLASE


class Circle {
  /** El radio del circulo */
  double radius = 1.0;

  /** Constructor del objeto circulo */
  Circle() {
  }

  /** Constructor del objeto circulo */
  Circle(double newRadius) {
   radius = newRadius;
  }

  /** Regresa el area del circulo */
  double getArea() {
    return radius * radius * Math.PI;
  }
}
