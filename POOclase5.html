Declaración de métodos get y set


El prefijo get se usa para contruir un método que retorne datos:




public class Estudiante {
   private String nombre;
   private int calificacion;

   public int getCalificacion() {
      return calificacion;
   }

}




El prefijo set se usa para contruir un método que asigne valores a los campos
de datos globales:






public class Estudiante {
   private String nombre;
   private int calificacion;

   public int getCalificacion() {
      return calificacion;
   }

   public void setCalificacion(int calificacion) {
      this.calificacion = calificacion;
   }
}

CÓMO SE NOMBRA UN PROGRAMA EN JAVA



El nombre del programa en Java se forma de la clase
principal seguido de la extensión java.



public class Estudiante {
   private String nombre;
   private int calificacion;

   public void setCalificacion(int calificacion) {
      this.calificacion = calificacion;
   }

   public int getCalificacion() {
      return calificacion;
   }
   
   public void setNonbre(Strig nombre) {
      this.nombre = nombre;
   }

   public String getNombre() {
      return nombre;
   }
   
   public static void main(String[] args) {
      Estudiante e1 = new Estudiante();

      e1.setNombre("Aurelio");
      e1.setCalificacion(6);

      System.out.println("El estudiante " + e1.getNombre() + 
      " tiene como calificación: " + e1.getCalificacion());
   
   }
}




El nombre del programa es el nombre de la clase principal.



El nombre del programa en Java es el siguiente: "Estudiante.java"
