Definición de objeto


Un objeto representa una entidad en el mundo real
que se identifica con facilidad. Ejemplos de
 objeto: un estudiante, un escritorio, un círculo,
 un botón, etc.

Un objeto tiene una única identidad, estado, y
 comportamiento.


Definición de estado
de objeto


El estado de un objeto es conocido como su propiedad
o atributo y que es representado por un campo de
datos con su valor actual.

Ejemplo: el objeto circulo, tiene un campo de datos
radio, el cuál es una propiedad que caracteriza
al círculo.

En el caso del rectángulo, el estado del objeto es
el ancho y la altura, lo que caracteriza al
rectángulo.


Comportamiento del
objeto


También llamado su acción y es definido por un
método. Para invocar un método sobre un objeto
se le pregunta al objeto para que desempeñe su
acción.

Ejemplo: para un círculo getArea() se invoca
para regresar el área.


Denotación de objeto


claseNombre objetoRefVar;

claseNombre objetoRefVar = new claseNombre();


Clases


Los objetos del mismo tipo son definidos usando
una clase común. Una clase es una plantilla, plano o 
contrato que define que el campo de datos de un
 objeto y métodos se realicen. Un objeto es una
 instancia de un clase.

Crear una instancia se refiere como
instanciación. El termino objeto e instancia
puede ser intercambiable.


Ejemplo de Clase


class Circle {
  /** El radio del circulo */
  double radius = 1.0;

  /** Constructor del objeto circulo */
  Circle() {
  }

  /** Constructor del objeto circulo */
  Circle(double newRadius) {
   radius = newRadius;
  }

  /** Regresa el area del circulo */
  double getArea() {
    return radius * radius * Math.PI;
  }
}

Definición de paradigma


1. Ejemplo o ejemplar.

2. Teoría o conjunto de teorías cuyo núcleo central
 se acepta sin cuestionar y que suministra la base y
 modelo para resolver problemas y avanzar en el
 conocimiento.
 El paradigma newtoniano.

3. Relación de elementos que comparten un mismo
 contexto fonológico, morfológico o sintáctico
 en función de sus propiedades lingüísticas.

4. Esquema formal en el que se organizan las
 palabras que admiten modificaciones flexivas
 o derivativas.

paradigma derivativo

1. Gram. Conjunto de palabras que comparten la
 misma raíz.

paradigma flexivo

1. Gram. Conjunto de formas que se obtiene con las
 variantes flexivas de una palabra; p. ej., alto,
 alta, altos, altas.

public class TestCircle1 {
  /** Metodo principal */
  public static void main(String[] args) {
    // Crea un circulo con radio 1.0
    Circle1 circle1 = new Circle1();
    System.out.println("El area del circulo de raio "
	+ circle1.radius + " es " + circle1.getArea() );
    // Crea un circulo con radio 25
    Circle1 circle2 = new Circle1(25);
    System.out.println("El area del circulo de radio "
	+ circle2.radius + " es " + circle2.getArea());
    // Crea un circulo de radio 125
    Circle1 circle3 = new Circle1(125);
    System.out.println("El area del circulo de raio "
	+ circle3.radius + " es " + circle3.getArea());
    // Modifica el radio del circulo
    circle2.radius = 100;
    System.out.println("El area del circulo de radio "
	+ circle2.radius + " es " + circle2.getArea() );
    }
  }
// Define la clase circulo con dos constructores
class Circle1 {
  double radius ;
  /** Construir del circulo con radio 1 */
  Circle1() {
    radius = 1.0;
  }
  /** Construir un circulo con radio especificado */
  Circle1(double newRadius) {
    radius = newRadius;
  }
  /** Regresa el area del circulo */
  double getArea() {
    return radius * radius * Math.PI;
  }
}
