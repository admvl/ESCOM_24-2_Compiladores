Pregunta 3                                                                                                                                       1 punto


¿Cómo se representa una gramática formal? Describa su simbología.


Respuesta. La GLC se representa como: G(S, N, T, P). Dónde, S representa el símbolo inicial, N representa los símbolos no terminales, T representa los símbolos terminales, y P representa el conjunto de producciones.


Pregunta 4                                                                                                                                       1 punto


¿Cómo se representa una derivación?


Respuesta. Las derivaciones son de dos tipos: derivación izquierda y derivación derecha. Se comienza la derivación con el símbolo inicial. Y esta se representa, como:


E ⇒ E + T ⇒ E + T * F ⇒ E + T * id ⇒
E + F * id ⇒ E + id * id ⇒ T + id * id ⇒
F + id * id ⇒ id + id * id

Pregunta 8                                                                                                                                       1 punto


¿Que significa LL(1)?


Respuesta. La primera L significa lectura de izquierda a derecha y la segunda L representa derivaciones a la izquierda y el 1 entre paréntesis significa un símbolo de análisis.


Pregunta 9                                                                                                                                       1 punto


¿Cuales son las condiciones para definir que una gramática es LL(1)?


Respuesta. La gramática debe cumplir con dos reglas, las cuales se aplican a producciones que tienen alternativas. Donde la regla UNO es, que la intersección de los primeros de cada una de las alternativas debe ser conjunto vacio. La regla DOS, se aplica cuando una de las alternativas es el caracter vacio, la cual, es que el primero del no terminal intersectado con el siguiente del mismo no terminal es un conjunto vacio.

¿Que es un elemento LR(0)?


Respuesta. Son las producciones de una gramática GLC que contienen un punto de análisis.





Pautas para la solución de los ejercicios


Procedimiento para la construcción de un analizador sintáctico:


Se obtiene una gramática libre de contexto de una cadena de tokens dada.
Se ajusta la gramática de acuerdo al tipo de analizador sintáctico que se va a construir.


Existen dos tipos de analizadores sintácticos, que son: analizadores sintácticos descendentes y los analizadores sintácticos ascendentes.


Los analizadores sintácticos descendentes usan gramáticas libres de contexto que deben ser del tipo LL(1).


Una gramática LL(1) debe cumplir con dos reglas.


Los analizadores sintácticos ascendentes tienen gramáticas del tipo LR().
