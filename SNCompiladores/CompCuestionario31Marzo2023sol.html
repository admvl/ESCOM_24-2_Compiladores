Pautas para la solución de los proyectos


Procedimiento para la construcción de un analizador lexico:


Se obtiene una expresión regular de los tokens de entrada.
Se clasifican usando las clases lexicas.


Se convierte la expresión regular en un AFN.
Se usan los modelos de Thompson de las operaciones básicas de las ER.


Se convierte el AFN en AFD.
Se usa el algoritmo de subconjuntos.


Se minimiza el AFD.
Se aplica la consistencia de los estados.


Se obtiene la tabla de transiciones δ[Estados, Tokens].


Se aplica el siguiente pseudo código para programar la tabla de transiciones δ[Estados, Tokens] :


      Tokens ← SiguienteToken();
      Estados ← s0 ;
      while Tokens <> $ || Estados <> se do
        Estados := δ[Estados, Tokens];
        Tokens := SiguienteToken();
      if Estados == sa then
        EsEnPan(Aceptado);
      else
        EsEnPan(Error);

Dónde: $ es el final, s0 es el estado inicial, se es un estado de error, y sa es un estado de aceptación.
