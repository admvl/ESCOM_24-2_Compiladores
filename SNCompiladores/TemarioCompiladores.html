Temario Compiladores del IPN-ESCOM, nivel II.

Prácticas sugeridas


Práctica 1: Diseño e implementación de las clases AFN, AFD.

Práctica 2: Diseño y desarrollo de un sistema para obtener el AFN asociado a una Expresión Regular por construcción de Thompson.

Práctica 3: Implementación del algoritmo para convertir un AFN a AFD.

Práctica 4: Desarrollo de un analizador léxico especificado por medio de expresiones regulares.



Práctica 5: Desarrollo de un analizador sintáctico por descenso recursivo.

Práctica 6: Diseño e implementación de la clase para obtener la tabla LL(1) de una gramática libre del contexto sin recursión izquierda.

Práctica 7: Diseño e implementación de la clase para obtener las tablas LR(0), LR(1), LALR.

Práctica 8: Generación de un analizador sintáctico ascendente para la gramática de un lenguaje de programación que soporte: Declaración de variables, estructuras de control, estructuras iterativas, expresiones relacionales y aritméticas, llamadas a funciones y procedimientos (Sugerencia HOC).

Práctica 9: Generación de código intermedio para la gramática del problema anterior.

Práctica 10: Diseño e implementación de algoritmos para optimización de código aplicados al código generado en la práctica 9.


Proyecto


Selecciona un bloque de código del lenguaje de alto nivel que quieras. El cuál se traducirá a un lenguaje de bajo nivel usando las etapas de un compilador. Este proyecto se implementará con un máximo de tres Estudiantes.



Forma de llevar el curso


1.- El contenido teórico se impartirá usando esta página, dónde se presenta el temario en el cuál se seleccionará el tema dando click en el enlace correspondiente.


2.- Las prácticas se enlistan en la sección guinda, las que se realizarán durante el curso conforme al avance teórico.


3.- Se presenta una lista de proyectos en la sección guinda. Los que podrán realizar en grupos de tres Estudiantes, como máximo.


4.- Se aplicarán tres examenes, cómo máximo.


Evaluación del curso


1.- Las prácticas y el proyecto sirven para acreditar el curso con 6.
