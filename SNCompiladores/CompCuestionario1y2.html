Pautas para solución de los proyectos


Procedimiento para la construcción de un analizador lexico:


Se obtiene una expresión regular de los tokens de entrada


Se convierte la expresión regular en un AFN


Se convierte el AFN en AFD


Se minimiza el AFD


Se obtiene la tabla de transiciones


Se aplica el siguiente código para programar la tabla de transiciones δ[Estados, car] :


      car ← SiguienteCaracter();
      Estados ← s 0 ;
      while car <> EOF ∧ Estados <> s e do
        Estados := δ[Estados, car];
        car := SiguienteCaracter();
      if Estados == s a then
        EsEnPan(Aceptado);
      else
        EsEnPan(Error);
