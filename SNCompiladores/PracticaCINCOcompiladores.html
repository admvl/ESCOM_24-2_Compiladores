Práctica 5

Temario

DESARROLLO DE UN ANALIZADOR SINTÁCTICO POR DESCENSO RECURSIVO

Objetivo


Construir las fuciones de todos los símbolos no terminales de una gramática libre de contexto.



Actividades


Se tiene la siguiente gramática libre de contexto:


Gramática libre de contexto:



S → M(P)C
M → p s v m | s v m | v m
P → Str [ ] args
C → { D }
D → t id ;


            Actividad 1. Acondicionar la gramática libre de contexto. (Presentar todo el proceso de acondicionamiento).


            Actividad 2. Presentar la tabla de producciones ya lista para comenzar a construir las funciones.


            Actividad 3. Presentar todas las funciones de los símbolos no terminales.


Cuestionario


Pregunta 1. Presentar un proceso para obtener las funciones de los no terminales de cualquier gramática libre de contexto.


Pregunta 2. Aplicando las funciones obtenidas de la Actividad 3, analizar la siguiente cadena:


psvm(Str[] args){int c;}




Sugerencias


Sugerencia 1. Para la Actividad 1 aplicar las reglas de las gramáticas LL(1) y si presentan problemas estas producciones, corregirlas eliminandoles el problema que se presente.


Sugerencia 2. Para la Actividad 2 explicar cómo se presentan la tabla de producciones, si es necesario aumentar la gramática.


Sugerencia 3. Para la Actividad 3 se debe declarar la función de lectura de los tokens de entrada. También se deben usar la tabla de primeros y siguientes.
