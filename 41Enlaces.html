Polimorfismo

Definición de polimorfismo del diccionario de la real acedemia de la lengua española. En Química. Propiedad de los elementos y sus compuestos, que pueden cambiar de forma sin variar su naturaleza.


Definición de polimorfismo en computación. Es cuando diferentes tipos de objetos responden de manera diferente a la llamada del mismo método.


Los términos polimorfismo y enlace dinámico están íntimamente relacionados, pero no son lo mismo. Es útil para conocer la diferencia. El polimorfismo es una forma de comportamiento. La vinculación dinámica es el mecanismo para eso comportamiento: cómo se implementa. Específicamente, el polimorfismo es cuando diferentes tipos de objetos responden de forma diferente a la misma llamada al método. El enlace dinámico es lo que hace la JVM para hacer coincidir un llamada al método polimórfico con un método particular. Justo antes de que la JVM ejecute una llamada a un método, determina el tipo de llamada real de la llamada al método. objeto. Si el objeto de llamada real es de la clase X, la JVM vincula el método de la clase X a la llamada del método. Si el objeto de llamada real es de la clase Y, la JVM vincula el método de la clase Y a la llamada del método. Después de la JVM enlaza el método apropiado a la llamada del método, la JVM ejecuta el método enlazado. Por ejemplo, tenga en cuenta la llamada al método obj.toString en la siguiente declaración:


System.out.println(obj.toString());

Enlace

Definición de enlace. Es el vínculo entre la llamada a un método y la definición del método. El enlace identifica que método se ejecuta. Existen dos tipos de enlaces: enlace estático y enlace dinámico.


Ejemplo de código de enlace

Código en java. El siguiente código es un ejemplo del vínculo entre el método y la definición del mismo:

        

 public class A {
    public void metodo1() {
      System.out.println("Metodo 1");
    }
    public void metodo2() {
      System.out.println("Metodo 2");
    }
    public static void main(string[] args) {
      A obj = new A();
      obj.metodo1 ();
      obj.metodo2 ();
    }
  }

Enlace estático

Definición de enlace estático. El compilador identificará el método que se va a ejecutar en tiempo de compilación. Generalmente el enlace estático se aplica cuando se usa sobrecarga.


Ejemplo de código de enlace estático

Código en java. Se presenta el código que incluye sobrecarga:


   public class Operacion {
     public suma(int x, int y) {
       System.out.println("La suma es:", x + y);
     }
     public suma(doble x, doble y) {
       System.out.println("La suma es:", x + y);
     }
     public static void main(string[] args) {
       Operacion cal = new Operacion();
       cal.sum(52,32);
       cal.sum(32.3,26.5);
     }
   }


Enlace dinámico

Definición de enlace dinámico. La identificación del método se hace en tiempo de ejecución. Generalmente se usa cuando se utilizan superclases.


Ejemplo de código de enlace dinámico

Código en java. Se presenta el código que incluye superclases:


   public class Forma {
     public dibujar() {
       System.out.println("Se dibuja una forma");
     }
   }
   public class Circulo extend Forma {
     public void dibujar() {
       System.out.println("Se dibuja un circulo");
     }
   }
   public class Triangulo extend Forma {
     public void dibujar() {
       System.out.println("Se dibuja un triangulo");
     }
   }
   public class Prueba {
     public static void main(string[] args) {
       Forma s;
       s = new Forma();
       s.dibujar();
       
       s = new Circulo();
       s.dibujar();

       s = new Triangulo();
       s.dibujar();
     }
   }
