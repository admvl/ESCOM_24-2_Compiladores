Declaración de métodos get y set


El prefijo get se usa para contruir un método que retorne datos:


public class Estudiante {
   private String nombre;
   private int calificacion;

   public int getCalificacion() {
      return calificacion;
   }

}



El prefijo set se usa para contruir un método que asigne valores a los campos
de datos globales:



public class Estudiante {
   private String nombre;
   private int calificacion;

   public int getCalificacion() {
      return calificacion;
   }

   public void setCalificacion(int calificacion) {
      this.calificacion = calificacion;
   }
}

class Circle1 {
  double radius ;
  /** Construir del circulo con radio 1 */
  Circle1() {
    radius = 1.0;
  }
  /** Construir un circulo con radio especificado */
  Circle1(double newRadius) {
    radius = newRadius;
  }
  /** Regresa el area del circulo */
  double getArea() {
    return radius * radius * Math.PI;
  }
}
